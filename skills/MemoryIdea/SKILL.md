A proposal — goal, alternatives, and potential approach for something not yet built. Lifecycle: Open → Exploring → Adopted (promoted via `/MemoryPromote`) or Dismissed (archived).

Read the template at the configured `template` path before creating the file. Replace Templater placeholders (`{{title}}`, `{{date:...}}`) with actual values.

Write to `memory/ideas/` using the title as filename. Tag with `type/memory/idea`. The `spark` field records what prompted the idea. The `idea` field is a concise description of the proposal. Status starts as `Open`.
